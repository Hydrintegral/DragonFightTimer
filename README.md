# DragonFight Timer

Times CubeKrowd dragon fights from the instant the dragon entity exists to the kill message. This mod is primarily intended for dragon fight hosts, but anyone is free to use it!

## Download

You can find any downloads in the **Versions** tab on this mod's [Modrinth page](https://modrinth.com/mod/ckdragonfighttimer).
